# 🖥️ Online Examination System - Java

A GUI-based Online Examination System developed using Java Swing and Object-Oriented Programming concepts.

---

## 📌 Project Overview

This project simulates an online examination platform where users can:

- Login using username and password
- Update profile details before starting the exam
- Attempt multiple-choice questions with a countdown timer
- Navigate between questions using Next and Previous buttons
- Submit the exam manually or automatically when time runs out
- View final score and exam result

---

## 🛠️ Technologies Used

- Java
- Java Swing
- Object-Oriented Programming (OOP)
- VS Code
